# AIDJ — 自然言語プロンプトライブラリ

各プロンプトは DJ が opencode TUI に入力する自然言語指示例。
エージェントは現在のシーンに応じて適宜解釈・適応すること。

## シーン / トランスポート

- 「次のシーンへ」— `active_scene` を +1 進める。scenes.yaml に BPM 宣言があればランプ。
- 「4 小節後にドロップ」— 次のフレーズ境界で予約投入して発火。
- 「ここで 16 小節キープ」— 次シーン投入を 16 小節遅延、エネルギー密度を下げる。
- 「テンポ倍に」— BPM を ×2、スイングは保持。
- 「テンポ半分に」— BPM を 1/2、キックをハーフタイムに。

## ドラム / ブレイク

- 「8 小節 amen を 2-step に切り替え」
- 「4 小節目にスネアラッシュ」
- 「ここで 4 小節キック抜き」
- 「220 BPM で amen の上に gabber キックを重ねて」
- 「2 小節にわたり 32 分ハイハット、ベロシティ減衰付き」
- 「全裏拍にゴーストスネア、1..127 のベロシティ交互」
- 「キックをヒューマナイズ: 4 小節にわたりベロシティ 90-127 で揺らす」

## ベース

- 「5 小節目からリーズ、サブベース寄りで」
- 「4 小節の off-beat リーズ、キックにサイドチェーン」
- 「C で 8 小節のワブルベース、LFO 1/8」

## リード / スタブ

- 「4 小節の裏拍にレイブスタブ、D マイナー」
- 「17 小節目から 16 分のア酸性 リード」
- 「次のドロップに向けてリードのフィルターを 0 -> 1 までスイープ」

## FX / マクロ

- 「ドラムのリバーブ送りを 60% まで開く」
- 「ブレイクにステラ感出す、 breakdown 中」
- 「16 小節かけてブレイクのビットクラッシュを 0 から 0.6 まで上昇」
- 「ベースのディストーションを 0.5 まで上げてドロップでリセット」

## クロストラック連携(conductor 宛て)

- 「33 小節目からハードコアへのトランジション、16 小節かけて BPM 180 -> 210」
- 「ブレイクダウン: 17 小節目でドラム + ベースをカット、パッド + リード残響のみ残す」
- 「49 小節目の指定: トラック 1 3 5 を伸ばす感じで」

## 複合 DJ 指示例

複数意図が混在する分類テスト用:

- 「次ドロップ: amen を gabber キックに差し替え、BPM 220 まで上げ、リーズ戻して、リバーブ送りを 30% にこする」
- 「今すぐブレイクダウン: 1 小節でキックとブレイクを落とし、パッドをフェードイン、BPM 140 に下げる」
- 「60 小節目から 64 小節目に向けてスネアロールを盛り上げ、ドロップで着地。リードブレイクはフィルターカットオフ 0 -> 1 で上昇」
